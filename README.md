# em-libcurl

[![MIT License](https://img.shields.io/badge/license-MIT-yellow.svg)](./LICENSE)

emeralds wrapper for [libcurl](https://curl.se/libcurl/)

# Installation

```sh
em install https://github.com/byfourthwall/em-libcurl
```
